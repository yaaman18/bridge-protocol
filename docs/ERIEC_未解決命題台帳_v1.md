## A. 構造内開問題(証明・反証で閉じる)

**A-1 橋関手 𝔉 の存在 — 解決済み(論理的存在、2026-07-06)** — `𝔉 : Δat → ℋat` であって `∀X, h₃(X) ⇔ Nontriv(𝔉(X))` を満たすものの存在。
`alphaRel` のみに依存する候補と非正規化 Gram `MᵀM` 候補には反例がある。一方、完全な `rho/sigma/kappa/epsilon` データから一次元 Hilbert 空間上の `id/0` を選び、これらを保存する構造同型を恒等 intertwiner に送る `structuralHingeHilbertFunctor` を構成した。その像で `h₃ ⇔ Nontriv` が成立する。従って論理的存在問題は閉じた。ただしこの関手は蝶番真偽を用いる classifier であり、物理 Jacobian から生成される標準的・計量的な橋ではない。その強化は別命題である。
証拠: `ERIEC.StructuralHinge.CompleteHingeIso.hingeWitness_iff`, `structuralHingeHilbertFunctor_nontrivial_iff`。反例境界は `ERIEC.BridgeFunctor.no_alphaOnly_hinge_classifier`, `raw_gram_bridge_counterexample`。
下位階層(要求強度順、どこまで登るかも未決): (a) 関手の存在 → (b) 随伴対 `𝔉 ⊣ 𝔊` → (c) 圏同値 → (d) 双解釈可能性。

**A-2 閉包関手 𝔊 の存在 — 解決済み(2026-07-06)** — `𝔊 : 𝒪at_V → Δat` であって `V`-不変性を `h₁∧h₂` の段階族へ送るもの。
反射的後続閉包 `K ↦ K ∪ post(K)` を左右二系統の定数段階族として構成した。`ViableSystem.step_closed` から viability 集合が両閉包の固定点となる。`pi/alpha` を恒等関係、`rho/sigma` を反射的一歩後続関係とする実フレームでは `Φ=T′=K∪post(K)`。viability・遷移を双方向保存する構造同型を `pi/rho/alpha/sigma/kappa/epsilon` の全フィールドを保存するフレーム同型へ送る `viabilityRelationalFunctor` を構成した。
証拠: `ERIEC.ViabilityClosure.viabilityRelationalFrame_left_fixed`, `viabilityRelationalFrame_right_fixed`, `viabilityRelationalFunctor_kappa_natural`, `viabilityRelationalFunctor_epsilon_natural`。

**A-3 三圏合成図式の可換性 — 解決済み(情報保存ループは否定、2026-07-06)** — 連続層→関係層→認証層→結合層→連続層のループが(A-1, A-2 の関手のもとで)可換か。
仕様と実装に存在する `viable open system → 関係フレーム → 蝶番データ → Hilbert intertwiner` の経路を `viableToHilbertFunctor` として合成し、二つの親括弧が一致すること、および像の `WldNontrivial` が viability 領域の非空性と同値であることを証明した。しかし二状態viable systemの恒等自己同型とBool反転自己同型が同一Hilbert恒等射へ潰れるため、この合成は faithful でなく、そのHom写像に左逆は存在しない。従って情報保存的な恒等閉ループは不可能。仕様にない任意の `Hilbert → open system` 戻り脚については主張しない。
証拠: `ERIEC.LayerComposition.viableToHilbert_comp_assoc`, `viableToHilbert_nontrivial_iff`, `viableToHilbert_not_faithful`, `no_bool_hom_left_inverse`。

**A-4 臨界減速(予想 6.9)** — 崩壊接近時 `λ_max(L_t) → 1`、回復時定数がスペクトルギャップ逆数で発散。
出典: v2.1 予想 6.9。解決の形: 離散崩壊(定理 5.4)と連続動態を結ぶ力学定理。依存: I-1(層間の動態対応なしには「接近時」が定義できない)。地平: 遠。実装フェーズ IV の最難関(v5 §13-8)と同根。

**A-5 表現選択の自然性 — 解決済み(2026-07-06)** — 有限関係同型上で接続行列の厳密なreindex自然性を証明。一般の前向関係射上では成分ごとのlax保存のみを証明し、厳密自然性への有限反例を構成した。
証拠: `formal/ERIEC/InterfaceLinearization.lean` の `relationMatrix_reindex_natural`, `relationMatrix_lax_natural`, `strict_naturality_fails_for_relationHom`。結論: A-1 の標準候補は関係同型のgroupoid上で厳密関手的、一般関係射上ではlax関手的に弱化する必要がある。

**A-6 蝶番分岐の余帰納核への遺伝(単段)** — `DC(𝔽,s)` のもと、蝶番上の分岐点(`m ∈ Act` かつ `2 ≤ |α(m)|`)が一つでもあれば `α(m) ⊆ νΘ`(系: `2 ≤ |νΘ|`)。分岐=非退化性の原子が余帰納核へ無条件に遺伝するという、豊かさの「増幅」側の単段定理。
出典: 2026-07-08 設計議論。E1 不採用決定(公理メモ §11-2)に伴い、対象層の豊かさは公理でなく「分類子 [DEF] + 条件付き増幅定理 + 証人義務」で扱う方針の第一便。証明素材の見込みは単調性(v5.1 定理1.1)+余帰納原理(定理3.2(i))のみで hConv 非依存(スケッチ: `m ∈ Act ⊆ σ★(ε(s))` より `α(m) ⊆ Θ(ε(s))`、`h₂` と単調性より `ε(s) ∪ α(m)` が後不動点、余帰納原理で `νΘ` に入る)。
解決の形: `Branch` 述語の定義(VP-RICH-001)と本定理(VP-RICH-002)の Lean 証明。階付き一様版と世代間保存版(`upd` による `Branch ∩ Act` の維持)は別命題として未起票。依存: なし(既存 `ERIEC.Closure` / `ERIEC.Hinge` / `ERIEC.DC` のみ)。地平: 近。

## M. メタ理論(体系についての定理)

**M-1 非退化モデルの任意規模存在定理** — `AX_core` の連言が、任意に大きい台・真に多価な関係・非自明な `|H|` を持つモデルを許すこと(融合定理)。有限 DC 参照モデルの任意サイズ fragment は `VP-M1-001` / `reference_models.arbitrarily_large_dc` として certified。v5.1 `AX_core = AX_stable ∪ AX_dyn` の離散・動態 Tier-1 fragment は `VP-M1-FULL-001a` / `reference_models.arbitrarily_large_ax_core_discrete` として certified。さらに World/Value 非自明性と §13.2 非退化参照義務を `DCWorldBridge` 型に束ねる Tier-2 shape fragment は `VP-M1-FULL-001b` / `reference_models.arbitrarily_large_three_layer` として certified。残る full fusion 強化は、無条件 DC↔World 同値や現象主張を含まないよう別途起票する。
解決の形: 構成的融合補題。**照応(人間規模の系との対応)の論理的前提条件**であり、これが偽なら理論は空クラスと照応することになる。依存: なし。地平: 中。M 類の先頭。

**M-2 第二剛性解析** — hGC で一度だけ行った「公理連言のモデル類特徴づけ」の体系化。対象: `Sig-2 ∧ R2′`(崩壊前提の連言が意図より痩せていないか)、`E5 ∧ 豊かな外的遷移`。
解決の形: 各連言についてモデル類の特徴づけ定理(非退化性の証明、または第二の剛性崩壊の発見)。依存: R-1, R-3。地平: 中〜遠。

**M-3 証明無関係性からの脱出** — 薄い圏・Prop レベルから profunctor・データ付き随伴・Set 上の真の終余代数への昇格。利得: [FLD]/[THM] 規律と §12 不変性がパラメトリシティ(自由定理)として自動化される可能性。
解決の形: 豊穣化された v3 仕様。地平: 遠。

**M-4 μ計算断片への内部化** — §5 の到達・安全・INS を μ計算の断片として再定式化し、決定手続きと計算量上界を定理として得る。凍結文書 §0 の FO(LFP)/FO(TC) 分類はこの前段。
解決の形: 断片の同定+標準結果の移入。依存: なし。地平: 遠。

**M-5 quantale 豊穣化による二層統合** — 離散関係と連続作用素を同一の豊穣的枠組みの二基底とみなし、I-1 を基底変換(change of base)として原理化する。
解決の形: v4 系仕様(大工事)。依存: I-1, M-3。地平: 最遠。二層分裂そのものを解消する唯一の既知経路。
