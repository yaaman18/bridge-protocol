# チャットパネルでインラインチャット相当の操作をするプロンプト集

Antigravity IDE で `Cmd+I` のインラインチャットが使えない間の代替プロンプトテンプレートです。
チャットパネルを開いて（`antigravity.toggleChatFocus` 等）以下のテンプレートをコピペして使ってください。

---

## 基本パターン

インラインチャットと同じ効果を出すには、**「どのファイルのどの部分を」「どうしたいか」** を明示するのがポイントです。

---

## 1. 選択範囲を編集する（最頻出）

インラインチャットで コードを選択 → `Cmd+I` → 指示 と同じ効果。

```
`src/adjunction.jl` の以下のコードを書き換えてください：

```julia
// ここに対象コードを貼り付ける
```

やりたいこと：[指示をここに書く]

書き換えたコードだけを出力してください。説明は不要です。
```

**例：**
```
`src/adjunction.jl` の以下のコードを書き換えてください：

```julia
function powerset(xs)
    items = collect(xs)
    n = length(items)
    result = Vector{Set{eltype(items)}}()
    ...
end
```

やりたいこと：ビット演算の代わりに再帰で実装してください。

書き換えたコードだけを出力してください。説明は不要です。
```

---

## 2. バグ修正

```
`[ファイルパス]` のこのコードにバグがあります：

```julia
// 問題のあるコードを貼る
```

エラー内容：[エラーメッセージや症状]

修正したコードだけを出力してください。
```

---

## 3. カーソル位置に新しいコードを追加する

インラインチャットで空行にカーソルを置いて生成する場合と同じ効果。

```
`src/adjunction.jl` の `sigma_star_induced` 関数の直後に、
以下の仕様を満たす関数を追加してください：

仕様：[ここに仕様を書く]

追加するコードだけを出力してください。
```

`src/adjunction.jl` のこの関数を説明してください：

```
```

変更後のコードだけを出力してください。
```

```

## Tips

- **「説明は不要です」「コードだけ出力」** と添えるとインラインチャットに近い出力になる
- **ファイルパスを必ず書く** ことで AI が文脈を正確に把握できる
- 長い関数は **関数名と行番号だけ書いて「ファイルを参照して」** と指示しても動く
  ```
  `src/adjunction.jl` の `check_K3` 関数（42〜49行目）を...
  ```
- Antigravity IDE が **`#file:` 参照**をサポートしている場合は `#file:src/adjunction.jl` と書くとファイルを自動で添付できる

---

## Cmd+I の代替キーバインド設定

`~/Library/Application Support/Antigravity IDE/User/keybindings.json` に以下を追加すると
`Cmd+I` でチャットパネルのフォーカスを切り替えられます：

```json
{
    "key": "cmd+i",
    "command": "antigravity.toggleChatFocus",
    "when": "editorFocus"
}
```
