# §14 凍結文書 v1 ― 賭けの述語論理形

本書は『§14 凍結準備書 v1』の実行成果物である。§14 の散文六項を閉論理式 `W₁–W₆` として凍結し、各文の独立性を有限証人模型で確立する。本書の批准後、v2.1 §14 の散文は非規範的注釈となり、規範は本書の式に移る。

## §0. 背景論理と論理的分類

**背景論理**: 多ソート一階論理。ソートは `S, A, E, C, W` を有限、`Ā = ℝ^A, Ē = ℝ^E` を実線形ソートとする。集合変数(`κ(s)` 等)は有限ソート上の一項述語として扱う。

**不動点の扱い**: `νΦ_w` への所属は最大不動点演算子を要するため、凍結文の母言語は正確には **FO(LFP)**(不動点付き一階論理)である。ただし全ソート有限のため、各模型上では純一階式に展開され(不動点は `|C|` 回の反復で安定)、全証人検査は多項式時間のモデル検査に落ちる。`[THM]`(標準事実)

**各凍結文の論理的分類**:

| 文 | 断片 | 決定性(有限模型上) |
|---|---|---|
| `W₁, W₂, W₃` | FO(LFP) + 無解釈述語 | 述語の解釈を与えれば PTIME 検査 |
| `W₄` | 量化子有界 FO | PTIME |
| `W₅` | FO + 濃度計数 | PTIME |
| `W₆` | FO(TC)(推移閉包) | PTIME(定理 2.1 の標準形による) |

## §1. 前提定義(v2.1 本体への追加分)

**定義 1.1(生核)** `RawCore(A,E,C,S) := (α, σ, π, ρ)`(整合フィールドを持たない四関係組)。`[DEF]`

**定義 1.2(整合成立)** `ConvHolds(κ₀ : RawCore) :≡ (∀a∀e, e∈α(a) ↔ a∈σ(e)) ∧ (∀a∀c, c∈π(a) ↔ a∈ρ(c))`。`[DEF]`(`ConvHolds(κ₀)` の成立は `κ₀` が `𝕂` に持ち上がることと同値。)

**基本述語の一階展開**(以降の略記の定義。すべて `Σ_core` 上の式):

```text
InH(a,s)   :≡ [∃c (c ∈ κ(s) ∧ a ∈ ρ_{ω(s)}(c))] ∧ [∃e (e ∈ ε(s) ∧ a ∈ σ_{ω(s)}(e))]
h₁(s)      :≡ ∀c [ c ∈ κ(s) → ∃a ( (∃c' (c' ∈ κ(s) ∧ a ∈ ρ_{ω(s)}(c'))) ∧ c ∈ π_{ω(s)}(a) ) ]
h₂(s)      :≡ ∀e [ e ∈ ε(s) → ∃a ( (∃e' (e' ∈ ε(s) ∧ a ∈ σ_{ω(s)}(e'))) ∧ e ∈ α_{ω(s)}(a) ) ]
h₃(s)      :≡ ∃a InH(a,s)
h₄(s)      :≡ ∃c (c ∈ κ(s) ∧ c ∈ D)
DC(s)      :≡ h₁(s) ∧ h₂(s) ∧ h₃(s) ∧ h₄(s)
c ∈ νΦ_w   :≡ [gfp Y ⊆ C. π_w★(ρ_w★(Y))](c)                    (FO(LFP))
Nontriv    :≡ ∃x ∈ Ā ( x ≠ 0 ∧ L(x) = x )
Pos𝐕(s,e)  :≡ ∃c ( c ∈ νΦ_{ω(s)} ∧ ∃a ( a ∈ σ_{ω(s)}(e) ∧ c ∈ π_{ω(s)}(a) ) )
```

**定理 1.3(正値の算術消去)** `s∈V` のもとで `0 < 𝐕_{ω(s)}(e) ↔ Pos𝐕(s,e)`。`[THM]`
> 証明: 定理 7.1(v2.1)により分母 `|νΦ| ≠ 0`。正値 ⇔ 分子の交差が非空 ⇔ `Pos𝐕`。∎
> これにより `W₂` は実数演算を含まない純関係式で凍結できる。

## §2. 凍結文(規範)

無解釈記号(公理ゼロ則): `Ph : S → Prop`、`Mat : S × E → Prop`、`impl : RawCore`、`k₀ : ℕ`。

```text
W₁(Ph)      :≡ ∀s ∈ S [ ( DC(s) ∧ Nontriv ) ↔ Ph(s) ]

W₂(Mat)     :≡ ∀s ∈ S [ DC(s) → ∀e ∈ E ( Pos𝐕(s,e) ↔ Mat(s,e) ) ]

W₃(Ph)  ‡   :≡ ∀s ∈ S [ ( ∃m ( InH(m,s) ∧ FM1(m,s) ∧ FM2(m,s) ∧ FM3(m,s) ∧ FM4(m) ) ) ↔ Ph(s) ]

W₄(impl)    :≡ ConvHolds(impl)

W₅(impl,k₀) :≡ ∀s ∈ S [ DC(s) → k₀ ≤ |{a : InH(a,s)}| ]

W₆          :≡ ∃ traj : ℕ → S [ ∀n ( traj(n) → traj(n+1) ) ∧ ∀n ∃m ≥ n, DC(traj(m)) ]
```

弱変種(併記、非規範): `W₁⁻, W₂⁻, W₃⁻` は各 `↔` を `→`(構造 → 現象/mattering の向き)に置換したもの。

**‡ 継承**: `W₃` のみ(`FM3` が再構成 `‡`、v2.1 定義 9.4)。`FM3` の批准・差し替え時は `W₃` を再批准する(規律 0.2-6)。

**定理 2.1(W₆ の有限標準形)** 有限 `S` 上で

```text
W₆  ↔  ∃s ( DC(s) ∧ TC⁺(→)(s,s) )
```

すなわち「DC 状態を通る閉路の存在」。`[THM]`
> 証明: (⇐) 閉路を永久周回する軌道。(⇒) 無限軌道は有限 `S` 上で DC 状態を無限回踏むから、鳩の巣によりある DC 状態 `s` を無限回訪れ、二訪問間の区間が `s →⁺ s` を与える。∎
> 系: `W₆` は FO(TC) 文であり、有限模型上で到達可能性計算により決定可能。

**注記 2.2(W₄ の正確化)** 剛性定理(v2.1 定理 1.3)により、旧随伴公理を `impl` に問う文は非退化域で恒偽であり凍結価値を持たない。`W₄` は修正後の正しい経験的問い「対象実装は双対整合か」を凍結する。`W₄` が真なら随伴は定理 1.4 により自動成立する。

## §3. 証人模型群

すべて有限であり、全検査は §0 の分類により機械的に決定可能。

### 3.1 𝕂✗(整合違反の生核 ― W₄ 偽証人)

`A={a}, E={e₁,e₂}, C=S=∅` 相当の最小台。`α(a)={e₁}, σ(e₁)=∅, σ(e₂)=∅, π, ρ` 空。

**補題 3.1** `¬ConvHolds(𝕂✗)`。`[THM]`
> 証明: `e₁∈α(a)` だが `a∉σ(e₁)`。∎

### 3.2 M⁰(極小模型 ― v2.1 義務 13.1 の実現)

`A={a}, E={e}, C={c}, S={s₀}, W={0,⊤}`。階数 0: `α(a)={e}, σ(e)={a}, π(a)={c}, ρ(c)={a}`。階数 ⊤: 全関係空。`κ(s₀)={c}, ε(s₀)={e}, D={c}, ω(s₀)=0`。

**補題 3.2** `M⁰ ⊨ AX_core` かつ `DC(s₀)`、`|{a : InH(a,s₀)}| = 1`、`Pos𝐕(s₀,e)`。`[THM]`
> 証明: 整合は構成(逆関係)による。反単調性: ⊤ で空。Sig-2: `Φ_⊤ ≡ ∅` より `w*=0<⊤`。`Φ₀({c})={c}` で `h₁`、`Θ₀({e})={e}` で `h₂`、`H₀={a}` で `h₃` と濃度、`h₄` 直接。`νΦ₀={c}` かつ `c∈π₀★(σ₀(e))` で `Pos𝐕`。∎

### 3.3 𝕂⁺(非退化核 ― v2.1 義務 13.2 の核部分)

`A={a₁,a₂}, E={e₁,e₂}, C={c₁,c₂}`。階数 0:

```text
α₀(a₁)={e₁,e₂}   α₀(a₂)={e₂}        (α は真に多価 ― 剛性定理の退化域外)
σ₀(e₁)={a₁}      σ₀(e₂)={a₁,a₂}     (逆関係: hConv 構成的充足)
π₀(a₁)={c₁}      π₀(a₂)={c₂}
ρ₀(c₁)={a₁}      ρ₀(c₂)={a₂}        (hConvP 構成的充足)
```

階数 ⊤: 全関係空(反単調・Sig-2 は 3.2 と同論法、`w*=0`)。

**補題 3.3** `Φ₀({c₁})={c₁}, Φ₀({c₂})={c₂}, νΦ₀=C`。`Θ₀({e₁})=Θ₀({e₂})=Θ₀(E)=E`。`H₀(C,E)={a₁,a₂}`。`[THM]`
> 証明: 直接計算(例: `ρ₀★(C)={a₁,a₂}`、`π₀★({a₁,a₂})=C`)。∎

### 3.4 M⁺(内的力学のみ ― W₁/W₃/W₅ の証人台、W₆ 偽証人)

台: `𝕂⁺`。状態 `S={s₀,s₁,s₂}`:

```text
s₀=(κ=C, ε=E, ω=0)   s₁=(C, E, ⊤)   s₂=(∅, ∅, ⊤)
δ(0,κ)=⊤ (κ≠∅)   δ(0,∅)=0   δ(⊤,·)=⊤
→ᵢ : s₀→s₁→s₂→s₂(内的更新則 v2.1 定義 5.1 の忠実な軌道)
→ₑ : なし        D={c₁}
連続層: Ā=ℝ², T=id, L=id, r(a₁)=(1,0), r(a₂)=(0,1)
FM データ‡: s₀ →_{a₁} s₁、o_ι(s₀)=0, o_ι(s₁)=o_ι(s₂)=1
```

**補題 3.4** `M⁺ ⊨ AX_core`。`V={s₀}` であり、`DC(s₀)`、`¬DC(s₁)`、`¬DC(s₂)`。`[THM]`
> 証明: 整合・反単調・Sig-2 は補題 3.3 系。R2′: `δ` は非減少かつ `κ≠∅∧ω<⊤` で厳密増加(定義通り)。E5: `→ₑ=∅` で空虚。`DC(s₀)`: 補題 3.3 の四項。`s₁`: `Φ_⊤(C)=∅⊉C` で `h₁` 崩壊。`s₂`: `κ=∅` で `h₃,h₄` 崩壊。軌道整合: `Ψ₀(C)=C∩C=C`, `Ψᴱ₀(E)=E`, `δ(0,C)=⊤` で `s₁`; `Ψ_⊤(C)=C∩∅=∅` で `s₂`; 以降不動。∎
> 注: κ は 2 歩で ∅ に達し、崩壊定理の上界 `|W|+|C|=4` と整合。R2′ は公理としてでなく具体的 `δ` の検証済み性質として成立している。

**補題 3.5(Conscious 証人)‡** `Conscious(a₁, s₀)` が成り立つ。`[THM]‡`
> 証明: `InH(a₁,s₀)`: `a₁∈ρ₀(c₁)∧a₁∈σ₀(e₁)`。FM1: `Φ^{−a₁}₀(C)=π₀★(ρ₀★(C)∖{a₁})=π₀★({a₂})={c₂}⊉C`。FM2: `|α₀(a₁)|=2`。FM3‡: `s₀→_{a₁}s₁` かつ `o_ι(s₁)≠o_ι(s₀)`。FM4: `Wld₀(id)=ℝ²` への射影で `r(a₁)=(1,0)≠0`。∎

### 3.5 M↻(外的補給付き循環 ― W₆ 真証人)

`M⁺` に外的遷移 `s₀→ₑs₀, s₁→ₑs₀, s₂→ₑs₀` を追加。

**補題 3.6** `M↻ ⊨ AX_core`(E5 は両読みで成立)、かつ `DC(s₀) ∧ TC⁺(→)(s₀,s₀)`。`[THM]`
> 証明: 核は状態に依らず一定なので、核同型読みの E5 では全状態対が関係するが、全状態が `→ₑ` で `s₀` に到達し核は自明同型なので充足。Δ-同型読みでは `ω` が異なる状態対は無関係となり空虚に近く充足。閉路: `s₀→ᵢs₁→ₑs₀`。∎

## §4. 独立性定理

**定理 4.1(Indep W₁)** `(∃Ph, W₁(Ph)) ∧ (∃Ph, ¬W₁(Ph))`。`[THM]`
> 証明: 真側: `Ph := λs, DC(s)∧Nontriv`(自己充足、任意模型)。偽側: `M⁺` で `Ph := λ_.⊥`。`s₀` で左辺真(`DC(s₀)`、`Fix(id)=ℝ²≠0`)、右辺偽。∎

**定理 4.2(Indep W₂)** `(∃Mat, W₂(Mat)) ∧ (∃Mat, ¬W₂(Mat))`。`[THM]`
> 証明: 真側: `Mat := Pos𝐕`。偽側: `M⁰` で `Mat := λ_ _.⊥`。`DC(s₀)∧Pos𝐕(s₀,e)`(補題 3.2)で双条件が破れる。∎

**定理 4.3(Indep W₃)‡** `(∃Ph, W₃(Ph)) ∧ (∃Ph, ¬W₃(Ph))`。`[THM]‡`
> 証明: 真側: `Ph :=` 左辺自身。偽側: `M⁺` で `Ph := λ_.⊥`、補題 3.5 により `s₀` で左辺真。`‡` は FM3 定義から継承。∎

**定理 4.4(Indep W₄)** `(∃impl, W₄(impl)) ∧ (∃impl, ¬W₄(impl))`。`[THM]`
> 証明: `impl := 𝕂⁺` の生部分(構成的充足)/ `impl := 𝕂✗`(補題 3.1)。∎

**定理 4.5(Indep W₅、`k₀=2`)** `(∃impl, W₅(impl,2)) ∧ (∃impl, ¬W₅(impl,2))`。`[THM]`
> 証明: 真側: `M⁺`(`V={s₀}`、`|H(s₀)|=2`)。偽側: `M⁰`(`|H(s₀)|=1<2`)。一般の `k₀≥2` には `A` を `k₀` 元に拡げた同型構成の族で対応(義務: `RefModel` にパラメタ族として登録)。∎

**定理 4.6(Indep W₆)** `AX_core` のモデルで `W₆` が真のもの(`M↻`)と偽のもの(`M⁺`)が存在する。`[THM]`
> 証明: 真側: 補題 3.6 と定理 2.1。偽側: `M⁺` の遷移グラフ `s₀→s₁→s₂→s₂` に DC 状態(`s₀` のみ)を通る閉路は存在しない。∎
> 注: 準備書 §4 は偽側を崩壊定理 5.4‡ に帰着させる計画だったが、具体的 `δ` の直接検証により **`W₆` の独立性は ‡ 非依存**に格上げされた。

## §5. 集約保存性定理

**定理 5.1(保存性)** 各 `i∈{1,…,6}` について、`AX_core` の有限モデルと解釈の対 `(𝔐ᵢ⁺, 𝔐ᵢ⁻)` が存在して `𝔐ᵢ⁺ ⊨ Wᵢ` かつ `𝔐ᵢ⁻ ⊨ ¬Wᵢ`。したがって一階論理の健全性により

```text
AX_core ⊬ Wᵢ    かつ    AX_core ⊬ ¬Wᵢ        (i = 1,…,6;  i=3 は FM3‡ に相対的)
```

すなわち §14 の賭けはいずれも現行公理群から決定されない。`[THM]`(定理 4.1–4.6 の集約)

**定理 5.2(前方安定性)** `Ph, Mat, impl, k₀` に言及しない任意の公理・定義の追加パック `A` について、§3 の証人が `A` のモデルであり続ける限り、定理 5.1 は `AX_core ∪ A` に対してそのまま成立する。`[THM]`
> 証明: 証人模型の再検査のみが要件であり、無解釈記号は `A` に出現しないため真偽の振り分けは不変。∎
> 運用: これが規律 0.2-8-3 の CI 検査(証人の型検査維持)の論理的内容である。
