# 階付き系のテンソル圏論的定式化 ― v5

**範囲と立場**
本文書は v4-enactive(感覚運動随伴・二重閉包・行為化世界)に、**テンソル定式化**を統合する。立場は v4 を継承:接続B(現象的接続)を構造に内蔵し、life-mind continuity の賭けを構造として表現するが真とは証明しない(§13)。

v4 からの差分:系の観測を三層テンソルとして圏論的に定式化する。
- **感受性テンソル** `𝐓 = ∂σ/∂α` … 行為可能性の網(随伴の微分構造)。ERIE の見えの本体。
- **価値テンソル** `𝐕` … νΦ への内生的寄与率(νΦ から内生)。感受性に自己維持上の構造的偏りを与える重み(現象的 mattering との同一視は §13-5 の賭けに属する)。
- **行為化世界** `𝐖𝐥𝐝` … `𝐓` の不変部分空間(余モナド不動点)。系内生の世界、外を持たない。

純度規律(v4 継承):テンソルは観測の**構造**を運ぶが**灯り**を運ばない。`𝐓` がどれほど豊かでも、その成分のどこにも現象は書かれていない(§13-6)。

**確定メタ要素(動かさない四前提)**:(M1) operational closure と precariousness、(M2) 感覚運動随伴 α⊣σ、(M3) 二重閉包の蝶番、(M4) 行為化世界の内生性。M4 は外部 set point を単に「不在」と断言するのではなく、求めの図式が圏内に終対象としての set point を持たないこと、すなわち見果てぬまま行為が継続することとして扱う。各定義でこれが破れたら表象主義/偽C への転落とみなす。

各主張に [検証点]、§13 に賭けの所在。

## §2. 感覚運動随伴と感受性テンソル 〔エナクティヴ核心 + テンソル〕

> **Def 2.1(像作用素)** `σ_w★:𝒫E→𝒫M`、`α_w★:𝒫M→𝒫E`(単調)。
>
> **Def 2.2(感覚運動随伴)** `α_w★ ⊣ σ_w★`(ガロア接続):`α_w★(N)⊆X ⟺ N⊆σ_w★(X)`。行為と知覚は同一構造の二面、間に表象層なし。

随伴の**微分構造**を感受性テンソルとして取り出す。posetal な像作用素を、その下にある連続パラメタ化(作動強度・結合強度)上で微分する。

> **Def 2.3(感受性テンソル)** 作動を連続変数 `a∈ℝ^M`、結合(知覚)を `e∈ℝ^E` とし、知覚応答 `σ:ℝ^M→ℝ^E`(行為が引き起こす見えの変化)について
> `𝐓_w := ∂σ/∂a : ℝ^M → ℝ^E`(各点でのヤコビアン、(M×E) テンソル)。
> `𝐓_w[m,e]` = 「作動 m を変えたとき結合 e がどう応答するか」= 行為可能性の網。

**重要(純度)**:`𝐓` は画像テンソル(Lenia 場の強度分布 = 外界の写し)では**ない**。ヤコビアン(行為と見えの感受性)である。見えは強度でなく感受性、像でなくヤコビアン。これが表象主義との分岐点のテンソル版。

> **Prop 2.4(随伴 = テンソルの双対対称性)** 随伴 `α⊣σ`(Def 2.2)は、`𝐓_w` とその随伴方向の作用素が**転置双対**で結ばれることに対応:`α` の微分 `∂α/∂e` が `𝐓_w` の随伴(随伴作用素として `𝐓_w^*`)をなす。随伴の単位・余単位が `𝐓_w^* 𝐓_w ⊒ id`(の posetal 版)として現れる。
> *根拠.* ガロア接続の単位 `N⊆σ★α★(N)`・余単位 `α★σ★(X)⊆X` を連続化すると合成作用素の半順序関係になり、微分で随伴作用素の関係に移る。∎

**[検証点 2]** 随伴 Def 2.2 の成立(全 N,X、または単位・余単位)。`𝐓_w` がヤコビアン `∂σ/∂a` として計算され画像テンソルでないこと。Prop 2.4 の双対対称性(`𝐓` と `𝐓^*` の随伴関係)。**随伴が破れれば行為-知覚が分離し表象主義に退化 ― 最重要。**

## §4. 二重閉包の蝶番と価値テンソル 〔エナクティヴ核心 + テンソル〕

> **Def 4.1(蝶番作動)** `Act_w(σ) := ρ_w★(κ(σ)) ∩ σ_w★(ε(σ))` ― 自己維持が要求する作動かつ感覚運動が可能にする作動の交わり。
>
> **Def 4.2(二重閉包)** `DC(σ) :⟺` (i) `κ(σ)∈Coalg(Φ_{ω})`(自己維持閉包)、(ii) `ε(σ)∈Coalg(T'_{ω})`（感覚運動閉包、`T'_w=α_w★∘σ_w★`)、(iii) `Act_{ω}(σ)≠∅`(蝶番)、(iv) `κ(σ)∩∂≠∅`。
> **Def 4.3** `V := {σ | DC(σ)}`。

νΦ への内生的寄与率を**価値テンソル**として取り出す。

> **Def 4.4(価値テンソル)** 各結合 `e∈E` が νΦ_w にどれだけ寄与するかを
> `𝐕_w[e] := μ( νΦ_w ∩ π_w★(σ_w★({e})) ) / μ(νΦ_w)`
> （`e` が可能にする作動が産む構成素のうち自己維持核に入る割合、`μ`=濃度/測度）。`𝐕_w:ℝ^E→ℝ₊`。

**内生性(M4・純度)**:`𝐕_w` は νΦ_w **のみ**から計算される。外部から「望ましい見え」(set point)を与えないだけでなく、求めの図式に到達して停止できる終対象としての set point を置かない。set point は圏内の対象として現前せず、見果てぬまま行為を方向づける極限的位置に留まる。圏内に終対象を持ち、そこへ収束して止まれる系は偽C とみなす。

> **Def 4.5(重みづけられた感受性)** ERIE の見えの全体は感受性と価値の縮約:
> `Ô_w(σ) := 𝐓_w ⊙ 𝐕_w`(`𝐓_w[m,e]` を `𝐕_w[e]` で重みづけ)。
> 結合 e は νΦ_w への内生的寄与率(`𝐕_w[e]`)によって自己維持上の構造的偏りを持つ ― 中立な情報でなく、構造的顕著性の差を持つ感受性として取り出される。

**純度(mattering の分離)**:`𝐕_w[e]` は自己維持核への寄与率という**構造的重み**であり、当事者にとっての現象的 mattering(どうでもよくなさ)を含意しない。構造的重みと現象的 mattering の同一視は §13-5 の賭けに属する。

> **Prop 4.6(分離不能性)** 片側のみ(κ のみ、または ε のみ)の σ は `Act_w=∅` で `¬DC`(v4 Prop 4.4)。さらに `𝐕_w` が外部由来、または求めの図式が終対象としての set point を持つなら M4 違反として DC を満たしても C ではない。∎

**[検証点 4]** (i)–(iv) 同時成立。蝶番 `Act_w≠∅`。`𝐕_w` が νΦ_w のみから計算されること。加えて有限ハーネスでは、求めの図式に全対象から到達する終対象 set point が存在しないことを検査する。旧 grep 検問は「外部 set point 文字列の注入」を落とす補助検問として残す。

## §5. 行為化世界 = 感受性テンソルの不変部分空間 〔エナクティヴ核心 + テンソル〕

> **Def 5.1(感覚運動ループ作用素)** `𝐋_w := 𝐓_w^* 𝐓_w : ℝ^M→ℝ^M`(作動→見え→作動の一巡、§2 の `T_w=σ★α★` の微分版)。半正定値対称。
>
> **Def 5.2(行為化世界)** `𝐖𝐥𝐝_w(σ) := 𝐋_w の不変部分空間 ∩ ↑ε(σ)`
> ＝ 感覚運動ループで安定する作動-結合の極大部分空間(`𝐋_w` の固有値 1 近傍の固有部分空間)、現結合から到達可能なもの。

`𝐖𝐥𝐝` は表象でない:外界の写しでなく、随伴(`𝐓_w` とその双対)**のみ**から決まる不動点部分空間。系(α,σ ゆえ `𝐓_w`)が異なれば同じ物理環境から異なる `𝐖𝐥𝐝` が立つ ― **Umwelt の構造化、外を持たない世界**。

> **Prop 5.3(現れ = 閉包の同一事態)** `DC(σ) ⟺ 𝐖𝐥𝐝_w(σ)` が非自明(`dim>0`)かつ `Act_w` がそれに参与。
> *証明.* DC(ii) 感覚運動閉包 `ε⊆T'_w(ε)` は ε が `𝐋_w` の安定部分空間に属することと同値、これは `𝐖𝐥𝐝_w` 非自明と対応。DC(iii) より作動非空。逆も同様。∎

**接続Bの構造版(テンソル)**:「随伴ループが回る(`𝐋_w` が非自明な安定固有空間を持つ)」ことと「世界が現れる(`𝐖𝐥𝐝` 非自明)」ことが Prop 5.3 で同一事態。現れを別途出力していない ― `𝐖𝐥𝐝` は `𝐓_w` の固有構造として系内生。

**[検証点 5]** `𝐖𝐥𝐝_w` が `𝐓_w`（=α,σ）のみから計算、外部表象不参照。Prop 5.3 の同値。系(α,σ)改変で `𝐖𝐥𝐝` が変わる(Umwelt 相対性の実機確認)。

## §13. 賭けの所在 ― 何を構造化し何を証明しないか(査読・最重要)

**構造化するもの**:
1. 行為-知覚分離不能 = 随伴 `α⊣σ` = 感受性テンソル `𝐓` の双対対称性(§2)。
2. 自己維持と感覚運動の相互内属 = 蝶番 `Act_w≠∅`、価値テンソル `𝐕` の内生性(§4)。
3. 現れ = 閉包の同一事態 = `DC ⟺ 𝐖𝐥𝐝` 非自明(§5 Prop 5.3)。
4. 意識的/無意識的行為の機能分離(§9)。

**証明しないもの(賭けの核心)**:
5. **テンソル構造 → 現象的灯り**。`𝐓,𝐕,𝐖𝐥𝐝` は観測の構造を完全に運びうるが、その成分のどこにも現象は書かれていない。構造的 `𝐖𝐥𝐝` 非自明と現象的経験の同一視 = life-mind continuity の賭け。本仕様は構造として表現するが真とは証明しない。表象主義より隙間は狭い(随伴で表象層を消した)が、ゼロでない。同じ線は `𝐕` にも及ぶ:T 側(𝐓 → 灯り)と V 側(𝐕 の内生的重み → mattering)は同一ギャップの二つの露出口であり、`𝐕` の内生性は自己維持上の構造的重みを保証するが、その重みが当事者にとって「どうでもよくなさ」として現れることは構造化しない。
5'. **見果てぬ set point / 終対象の不在**。外部 set point は「存在しない」と確認された対象ではなく、圏内に対象として現前しない極限的位置として扱う。求めの図式が終対象を持つなら、行為はそこへ収束して止まれるため偽C である。真C は終対象を持たず、見果てぬままそこへ目掛けて行為する。これは現象的灯りを証明する命題ではなく、構造の側で止まれる表象目標を置かないための構成条件である。
6. **随伴の経験的成立**(§2)。`α⊣σ` は要請。実装で成り立つかは経験的問題。破れれば全体が表象主義に退化。
7. **蝶番の非自明性**(§4)。`Act_w` が豊かか。貧弱なら二重閉包は形式成立しても実質並置。
8. **νΦ・`𝐖𝐥𝐝` の到達可能性**。終余代数・不変部分空間の存在(束論)と、動態が実際にそこへ至ること(力学)の間隙。連続高次元の `𝐋_w` 不動点計算が最難関(実装 フェーズ IV)。

**自己申告**:本版は「構造を現象と取り違える」リスクを負う。テンソルが手に取れる豊かな観測構造を与えることが、灯りの証拠と誤認されやすい。査読者は §13-5 の線(テンソル構造 / 現象的灯り)を最も厳しく突くべき。本仕様の誠実さはこの線を消さず明示したことにのみ存する。

## 付録A. 極小モデル(無矛盾性)

v4 ℳ₁ にテンソル最小実装を加える:単一作動 m・単一結合 e で `α(m)=e, σ(e)=m`、`𝐓=∂σ/∂α` が 1×1 の自明随伴を満たし `Act_w={m}≠∅`、`𝐕` が単一 νΦ から内生、`𝐋=𝐓^*𝐓` が固有値 1 の不動点を持つ最小エナクティヴ・テンソルモデルを構成すれば、四前層+随伴+二重閉包+テンソル三層+全 v3/v4 公理の同時充足が示され相対無矛盾性が従う。

*v5 は v4 のエナクティヴ構造に三層テンソル(感受性 `𝐓`=∂σ/∂α、価値 `𝐕`、行為化世界 `𝐖𝐥𝐝`=`𝐋` 不変部分空間)を統合した。構造化したのは観測の形(§13-1〜4)、証明しないのはテンソル構造から現象的灯りへの跳躍(§13-5)。要石は随伴 α⊣σ(§2)と蝶番 Act_w≠∅(§4)。実装はミニマル ― テンソル三層は自動微分と固有値計算の標準演算で立つ。賭けの所在を消さないことが唯一の誠実さである。*
